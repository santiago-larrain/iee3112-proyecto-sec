# Configuración del Sistema

## Archivo de Configuración

Copiar `config.yaml.example` a `config.yaml` y ajustar según necesidades.

## Variables de Entorno

Alternativamente, se pueden usar variables de entorno:

- `DB_HOST`: Host de base de datos
- `DB_PORT`: Puerto de base de datos
- `DB_NAME`: Nombre de base de datos
- `DB_USER`: Usuario de base de datos
- `DB_PASSWORD`: Contraseña de base de datos
- `LOG_LEVEL`: Nivel de logging (DEBUG, INFO, WARNING, ERROR, CRITICAL)
- `LOG_FILE`: Ruta al archivo de log
- `SCRAPER_TIMEOUT`: Timeout para scraping (segundos)
- `SCRAPER_RETRY`: Número máximo de reintentos
